🎛️ Display settings (global)

These settings affect ALL agents. Individual agents can override their personal preference in their Profile settings.

🗂️ Disposition categories

Categories and subcategories agents choose from after each call. Sample data is provided — edit as needed. Use one line per category in the format: Category: sub1, sub2, sub3

🏷️ Custom AUX statuses

Add your own agent-selectable activity statuses (like Break or Lunch). Agents can pick these from their status dropdown. Custom statuses are treated as "away" — the agent is not in the call queue while on one. System statuses (Available, On Call, ACW, Not Available, End of Shift) cannot be changed.

Changes apply to all agents the next time their view refreshes.
